# KROSS - Telegram Mini App

## Описание
Мини-приложение для Telegram для чтения книги по главам.

## Файлы проекта

### Основные файлы:
- `index.html` - главная страница приложения
- `app.js` - логика приложения (загрузка глав, навигация, сохранение прогресса)
- `style.css` - стили приложения с поддержкой светлой и тёмкой темы Telegram

### Файлы глав:
- `chapter1.html` - Глава 1
- `chapter2.html` - Глава 2 (заглушка - добавьте содержимое)
- `chapter3.html` - Глава 3
- `chapter4.html` - Глава 4
- `chapter5.html` - Глава 5
- `chapter6.html` - Глава 6
- `chapter7.html` - Глава 7

## Функциональность

### Основные возможности:
✅ Чтение 7 глав книги
✅ Навигация между главами (кнопки, свайпы, стрелки)
✅ Сохранение прогресса чтения (Telegram CloudStorage)
✅ Поддержка светлой и тёмкой темы Telegram
✅ Адаптивный дизайн для мобильных устройств
✅ Плавные анимации переходов между главами

### Управление:
- **Кнопки 1-7** вверху экрана - быстрый переход к главе
- **Стрелки ◀ ▶** внизу - переход к предыдущей/следующей главе
- **Свайп влево/вправо** - навигация между главами
- **Прокрутка** - чтение текущей главы

## Установка и запуск

### Для локального тестирования:
1. Разместите все файлы в одной папке
2. Откройте `index.html` в браузере

### Для Telegram Mini App:
1. Загрузите все файлы на веб-хостинг (с HTTPS)
2. Создайте бота через @BotFather
3. Настройте Mini App URL через команду `/newapp`
4. Укажите URL вашего хостинга

## Особенности реализации

### Сохранение прогресса:
- Использует Telegram CloudStorage API
- Автоматически сохраняет номер текущей главы
- При повторном открытии загружается последняя прочитанная глава

### Темы:
- Автоматическое переключение между светлой и тёмкой темой
- Использует цветовую схему Telegram
- Переменные CSS: `var(--tg-theme-*)`

### Анимации:
- Плавное появление глав при переходе
- Анимация кнопок при нажатии
- Плавная прокрутка

## Доработка

### Чтобы добавить содержимое главы 2:
1. Откройте `chapter2.html`
2. Вставьте текст главы в тег `<body>`
3. Сохраните файл

### Чтобы добавить больше глав:
1. Создайте файл `chapter8.html` (и далее)
2. В `app.js` измените `const totalChapters = 7;` на нужное число
3. В `index.html` добавьте кнопки для новых глав

## Требования
